# AI Concierge Chatbot

## Setup

1. Clone the repository
```bash
git clone https://github.com/angel-puertas/ai-concierge-chatbot.git
```

2. Install dependencies
```bash
cd backend
npm install
```

```bash
cd frontend
npm install
```

3. Configure environment variables
```bash
cd ../backend
```

```bash
cp .env.example .env
```

## Usage
4. Run the server
```bash
cd ../backend
npm run dev
```

5. Run the frontend
```bash
cd ../frontend
npm run dev
```
